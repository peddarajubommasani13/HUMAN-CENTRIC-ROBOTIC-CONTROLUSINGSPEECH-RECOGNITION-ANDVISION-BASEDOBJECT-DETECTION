# Human-Centric Robotic Control Using Speech Recognition and Vision-Based Object Detection

## 📌 Project Overview
This project presents a human-centric robotic control system that combines Speech Recognition and Computer Vision techniques for intelligent robotic automation. The robot can identify, pick, and place colored objects into fixed locations using voice commands and real-time object detection.

The system uses OpenCV for object detection and tracking, Speech Recognition for voice-controlled operations, and robotic movement control for automated object handling.

---

## 🚀 Features
- Voice-controlled robotic actions
- Real-time object detection using OpenCV
- Color-based object identification
- Automated object picking and placement
- Human-robot interaction system
- Real-time camera processing

## 🎯 Working Process
1. User gives voice command (Pick Red / Green / Blue)
2. Camera captures live video feed
3. OpenCV detects the colored object
4. System calculates object coordinates
5. Robot moves to the object location
6. Object is picked using suction mechanism
7. Robot places the object in a predefined location
